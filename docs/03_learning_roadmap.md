# Learning Roadmap

## Beginner

- Transformer block intuition
- Basic trace/span mental model
- Local telemetry pipeline and storage
- Simple latency histograms

## Intermediate

- Async workers and queue semantics
- Retry and backoff patterns
- Continuous batching fundamentals
- Token streaming flow

## Advanced

- KV cache behavior and tradeoffs
- Quantization simulation and performance impacts
- Parallel inference strategies
- Observability architecture decomposition

## Expert

- Capacity planning and SLOs
- Fault isolation and resilience patterns
- Distributed serving tradeoffs (latency vs throughput)
- End-to-end optimization experiments
