# Beginner Onboarding Plan

## Week 1

- Setup local environment
- Run API + dashboard
- Generate first traces
- Inspect queue and latency metrics

## Week 2

- Add one trace field and propagate through pipeline
- Build one dashboard chart
- Write one analytics test

## Week 3

- Implement one optimization experiment (batch size vs p95)
- Document findings in notebooks

## Week 4

- Contribute a mini-module (e.g., failure simulator)
- Ship docs + tests + benchmark output
